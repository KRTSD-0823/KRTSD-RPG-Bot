# コンパイル時に実行するコマンドについて
package.jsonを見れば分かると思いますが、`npm run`で使えるコマンドをまとめておこうと思います。
- test：srcフォルダー内のmain.tsを直接実行。コンパイルは無し。
- start：buildフォルダーのコンパイルされたmain.jsを実行。コンパイルは無し。
- compile：src内のファイルおよびフォルダーをコンパイルする。
- all：`conpile`と`start`を合わせたもの
